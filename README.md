## ðŸš€ How to Use

First consult the prolog. After that use the `rodar` command.
```
?- rodar. 
```

Now you are playing the game. Use the following commands to act.
```
|: pegar(x). 
|: ir_para(x). 
```
Use `pegar(x)` to get `x` to your invertory and `ir_para(x)` to go to `x`.
